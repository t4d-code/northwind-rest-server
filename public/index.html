Northwind REST API Server

Welcome to the Northwind REST API Server

To access and modify resources, you can use any HTTP method: GET POST PUT PATCH DELETE OPTIONS

Examples

  • Get all categories: GET {{origin}}/api/categories
  • Get a category by Id 1: GET {{origin}}/api/categories/1
  • Adds a new category: POST {{origin}}/api/categories
  • Edit a category: PUT {{origin}}/api/categories
  • Delete a category with Id 1: DELETE {{origin}}/api/categories/1
